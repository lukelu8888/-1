## 系统概览

### 🎯 系统定位

**终极表单DIY工作台（Ultimate Form Designer Studio）** 是为福建高盛达富建材有限公司B2B外贸独立站量身打造的专业表单设计工具，采用台湾大厂风格UI设计，支持全业务流程表单的可视化设计与定制。

### ✨ 核心特性

- 🎨 **所见即所得设计**：实时A4画布预览，拖拽式布局
- 📋 **全业务表单覆盖**：询价、报价、合同、发票、装箱单、对账单
- 🎨 **专业样式定制**：颜色、字体、Logo自定义
- 💾 **版本历史管理**：撤销/重做、版本回溯
- 🔄 **表单模板管理**：复制、创建、导入、导出
- ⚡ **实时预览导出**：PDF导出、打印预览

### 🏢 支持的表单类型

| 表单类型 | 中文名称 | 英文名称 | 所属方 | 用途 |
|---------|---------|---------|-------|------|
| Inquiry | 客户询价单 | Customer Inquiry Form | Customer | 客户向Cosun询价 |
| Quotation | Cosun报价单 | Quotation | Cosun | Cosun向客户报价 |
| Sales Contract | 销售合同 | Sales Contract | Cosun | Cosun与客户签订 |
| Purchase Contract | 采购合同 | Purchase Contract | Cosun | Cosun向供应商采购 |
| Commercial Invoice | 商业发票 | Commercial Invoice | Cosun | 出口发票 |
| Packing List | 装箱单 | Packing List | Cosun | 货物装箱清单 |
| Statement | 对账单 | Statement of Account | Cosun | 月度对账 |

## 快速开始

### 🚀 方式一：快速访问页面

1. 在浏览器中打开 `ultimate-form-designer-quick-access.html`
2. 点击 **"立即开始设计"** 或 **"打开表单库"**

## 核心功能

### 1. 表单选择器（Select Template）

#### 功能说明
- 浏览所有可用的表单模板
- 查看模板基本信息（区块数、更新日期、创建人）
- 选择模板进入编辑模式
- 复制现有模板创建新版本
- 创建全新自定义表单

#### 操作步骤
```
1. 点击表单卡片上的"编辑"按钮 → 加载模板到设计器
2. 点击表单卡片上的"复制"按钮 → 创建模板副本
3. 点击右上角"创建新表单"按钮 → 从零开始设计
```

### 2. 设计布局（Design Layout）

#### 2.1 左侧属性面板

**基本信息设置**
- 表单名称（中文/英文）
- 版本号
- 表单类型

**页面设置**
- 页眉/页脚显示开关
- 页面方向（纵向/横向）
- 页边距设置

**Logo设置**
- Logo显示开关
- Logo位置（左对齐/居中/右对齐）
- Logo来源（客户/Cosun/供应商）

**标题设置**
- 标题文字
- 字体大小（12-32pt）
- 对齐方式（左/中/右）

#### 2.2 中间画布区域

**工具栏功能**
- ➕ **添加区块**：创建新的表单区块
- 🔲 **网格开关**：显示/隐藏背景网格
- 📏 **标尺开关**：显示/隐藏辅助标尺
- 🔍 **缩放控制**：50%-200%缩放

**A4画布**
- 实时预览表单布局
- 按照实际尺寸显示（210mm x 297mm）
- 支持纵向/横向切换
- 页眉、正文、页脚完整展示

**区块操作**
- 点击区块选中（显示蓝色边框）
- 区块标题编辑
- ➕ 添加字段到区块
- 🗑️ 删除整个区块

**字段操作**
- 点击字段选中（右侧显示编辑器）
- 实时预览字段类型
- 🗑️ 删除字段
- 拖拽调整位置（开发中）

#### 2.3 右侧字段编辑器

当选中某个字段时，右侧显示字段编辑器：

- **字段标签**：显示在表单上的名称
- **字段类型**：文本/数字/日期/多行文本/下拉选择/表格
- **字段宽度**：1/3宽度/1/2宽度/全宽
- **必填标记**：是否为必填字段
- **占位符文本**：输入提示文字

### 3. 样式设置（Style Settings）

#### 3.1 配色方案

**预设配色**
- 专业蓝：`#0066cc` / `#475569`
- 商务红：`#dc2626` / `#64748b`
- 财务绿：`#059669` / `#64748b`
- 物流橙：`#ea580c` / `#64748b`
- 紫罗兰：`#7c3aed` / `#64748b`
- 青色：`#0891b2` / `#64748b`

#### 3.2 自定义颜色

- **主色调**：用于标题、边框、强调元素
- **辅助色**：用于次要文字、辅助线条

#### 3.3 字体设置

- **字体系列**：Arial / Times New Roman / Courier New / Helvetica
- **基础字号**：8-16pt
- **行高**：1.0-2.0

### 4. 实时预览（Preview）

#### 功能特性

- 📄 **完整表单预览**：所见即所得的最终效果
- 🖨️ **打印预览**：调用浏览器打印功能
- 📥 **导出PDF**：一键生成PDF文件（开发中）
- 📋 **示例数据填充**：显示表格和字段的示例数据

#### 预览内容

- ✅ 页眉（Logo + 公司信息 + 标题）
- ✅ 所有区块和字段
- ✅ 签名栏（如果启用）
- ✅ 页脚（日期 + 表单编号 + 页码）

## 详细操作指南

### 🎯 场景一：编辑现有表单模板

**需求**：修改"Cosun报价单"的标题和Logo位置

```
步骤：
1. 在"选择模板"标签页找到"Cosun报价单"
2. 点击"编辑"按钮
3. 切换到"设计布局"标签页
4. 在左侧属性面板找到"标题设置"
5. 修改标题文字为"QUOTATION / 正式报价单"
6. 调整字体大小为22pt
7. 在"Logo设置"中将位置改为"居中"
8. 点击顶部工具栏的"保存"按钮
9. 切换到"实时预览"查看效果
```

### 🎯 场景二：创建新的自定义表单

**需求**：创建一个"质量检验报告"表单

```
步骤：
1. 点击右上角"创建新表单"按钮
2. 在左侧属性面板修改基本信息：
   - 表单名称：质量检验报告
   - 英文名称：Quality Inspection Report
3. 点击画布工具栏的"添加区块"
4. 修改区块标题为"检验信息"
5. 点击区块内的"添加字段"
6. 在右侧编辑器配置字段：
   - 字段标签：检验日期
   - 字段类型：日期
   - 必填标记：开启
7. 重复步骤5-6添加更多字段
8. 切换到"样式设置"选择配色方案
9. 切换到"实时预览"查看效果
10. 满意后点击"保存"
```

### 🎯 场景三：批量调整表单样式

**需求**：将所有表单统一使用"专业蓝"配色

```
步骤：
1. 在"表单库管理中心Pro"中查看所有表单
2. 逐个打开表单到"终极表单DIY工作台"
3. 切换到"样式设置"标签页
4. 点击"专业蓝"配色预设
5. 点击"保存"
6. 重复步骤2-5处理其他表单
```

### 🎯 场景四：导出表单为PDF

**需求**：将设计好的表单导出为PDF文件

```
步骤：
1. 在"设计布局"或"实时预览"页面
2. 点击顶部工具栏的"导出PDF"按钮
3. 等待生成（当前版本提示"开发中"）
4. 临时方案：点击"打印"按钮
5. 在打印对话框中选择"另存为PDF"
6. 选择保存位置，点击保存
```

## 常见问题

### ❓ 如何修改表单的页边距？

**答**：目前页边距是在表单模板的`pageSettings.margins`中预设的，暂不支持在UI中直接修改。您可以：
1. 导出表单JSON文件
2. 修改`margins`属性（单位：mm）
3. 重新导入

### ❓ 字段可以拖拽排序吗？

**答**：拖拽排序功能正在开发中。当前版本可以：
1. 删除字段
2. 重新添加到期望位置
3. 使用撤销/重做功能调整

### ❓ 如何备份我的表单设计？

**答**：
1. 点击"导出JSON"按钮下载表单配置
2. 文件自动保存到浏览器下载目录
3. 文件名格式：`{formId}_{version}.json`
4. 需要恢复时，可以通过"导入"功能加载

### ❓ 表单是否支持多语言？

**答**：当前版本的表单是中英文双语设计：
- 字段标签同时包含中英文
- 可以在字段编辑器中修改
- 表单标题支持自定义

### ❓ 如何添加公司Logo？

**答**：
1. 在"设计布局"页面左侧属性面板
2. 找到"Logo设置"
3. 开启"显示Logo"开关
4. 选择"Logo来源"（客户/Cosun/供应商）
5. 调整Logo位置和大小

**注意**：实际Logo图片的上传功能正在开发中，当前显示占位符。

### ❓ 导出的PDF质量如何保证？

**答**：
1. 使用浏览器打印功能时，选择"另存为PDF"
2. 设置打印质量为"高质量"
3. 确保缩放比例为100%
4. 专用PDF导出功能正在开发，将提供更高质量输出

### ❓ 表单数据如何与业务流程关联？

**答**：
- 表单设计完成后，可在"表单工作流集成中心Pro"配置
- 支持字段映射、自动填充、验证规则
- 与WorkflowEditorPro中的业务流程深度集成

### ❓ 撤销次数有限制吗？

**答**：
- 默认历史记录保存最近50个版本
- 可以无限次撤销/重做
- 关闭页面后历史记录会清空
- 建议定期保存重要版本
