## 🎯 核心算法

### **1. 负载均衡算法**

```typescript
/**
 * 选择负载最低的业务员
 */
function getRecommendedSalesRep(region: Region): Personnel | null {
  // 1. 筛选同区域业务员
  const regionalSalesReps = personnelList
    .filter(p => p.role === '业务员' && p.region === region);
  
  // 2. 按负载排序（低→高）
  regionalSalesReps.sort((a, b) => (a.workload || 0) - (b.workload || 0));
  
  // 3. 返回负载最低者
  return regionalSalesReps[0] || null;
}
```

**示例：**
```
北美区业务员：
  王建: 负载3 🟢 ← 被选中
  张伟: 负载5 🟡
  李明: 负载7 🔴
```

### **2. 智能路由算法**

```typescript
/**
 * 查询或自动分配业务员
 */
function routeToSalesRep(customerName: string, customerRegion: Region): Personnel | null {
  // 步骤1: 查询已有关系
  const existing = getSalesRepByCustomer(customerName);
  if (existing) {
    return existing; // 老客户
  }
  
  // 步骤2: 新客户，自动分配
  const newSalesRep = getRecommendedSalesRep(customerRegion);
  
  // 步骤3: 保存新关系
  saveMappingToStorage(customerName, newSalesRep);
  
  return newSalesRep;
}
```
