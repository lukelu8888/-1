##### 6. 本地制造加工服务商（A级优先）⭐ 新增
**典型代表**：门窗加工厂、铝材加工商、窗帘制造商、家具定制厂、钣金加工厂、金属结构加工、玻璃深加工厂

**客户特征**：
- 🎯 **多品类需求**：不是单一产品，而是数十种原材料和配件
- 🔄 **持续性采购**：生产需要，每月/每周固定补货
- 📍 **本地化运营**：通常服务本地市场，对供应稳定性要求高
- 💰 **中等单笔金额**：单次采购金额不大，但累计年采购额可观
- 🤝 **长期合作价值**：一旦建立信任，合作周期长、复购率高

**关键信息收集**：
- 采购负责人联系方式（通常是老板或生产经理）
- 生产经理联系方式
- 加工类型（门窗/窗帘/家具/钣金/玻璃等）
- 主要产品线（定制门窗/成品窗帘/办公家具等）
- 月产能/年产值
- 主要客户群体（地产商/装修公司/零售商）
- 原材料需求清单（越详细越好）
- 月用量/年用量
- 当前供应商数量和痛点
- 付款周期偏好

**多品类原材料需求示例**：

**门窗加工厂**：
- 铝型材：门窗型材、幕墙型材（各种规格）
- 五金配件：铰链、拉手、月牙锁、多点锁、滑轮、滑轨
- 密封材料：密封条、毛条、泡沫条
- 玻璃配件：玻璃垫片、定位块、玻璃胶
- 紧固件：自攻螺丝、拉铆钉、膨胀螺栓
- 表面处理：粉末涂料、氟碳涂料
- 辅助材料：角码、加强筋、转接件

**窗帘制造商**：
- 轨道系统：铝合金轨道、罗马杆、电动轨道
- 配件：滑轮、挂钩、拉绳、吊环、窗帘杆托架
- 五金：窗帘圈、挂钩、夹子、磁吸
- 电机：窗帘电机、遥控器、智能控制系统
- 装饰件：窗帘头、花边、流苏、绑带
- 辅料：缝纫线、魔术贴、铅坠

**家具定制厂**：
- 五金配件：铰链、抽屉滑轨、拉手、锁具
- 连接件：三合一连接件、偏心轮、螺母
- 功能配件：气撑、阻尼器、缓冲垫、脚垫
- 装饰件：封边条、铝合金包边、装饰条
- 电气件：LED灯条、感应开关、插座
- 工具配件：安装工具、固定件

**钣金加工厂**：
- 原材料：冷轧板、镀锌板、不锈钢板、铝板
- 紧固件：螺丝、螺母、铆钉、焊钉
- 表面处理：喷塑粉末、油漆、电镀材料
- 辅助材料：焊材、切割片、砂纸、清洁剂

**玻璃深加工厂**：
- 辅材：玻璃胶、密封胶条、结构胶
- 配件：玻璃夹、驳接爪、点式配件、拉手
- 工具：玻璃刀、吸盘、切割工具
- 包装材料：保护膜、气泡膜、边角保护

**开发策略**：
- ✅ **一站式供应方案**：整合多品类产品，减少供应商管理成本
  - "您现在有多少家供应商？我们能帮您整合到2-3家"
  - "我们产品线覆盖门窗80%的配件需求"
  
- ✅ **统一账期管理**：
  - "统一月结账期，简化财务流程"
  - "批量采购享受更优惠价格"
  
- ✅ **本地化仓储服务**（如有）：
  - "紧急补货当天发货"
  - "提供VMI库存管理服务"
  
- ✅ **技术支持**：
  - "免费提供安装指导"
  - "新品培训和样品测试"
  - "技术问题快速响应"
  
- ✅ **小批量灵活起订**：
  - "理解加工厂多SKU小批量需求"
  - "支持混装拼柜，降低库存压力"
  
- ✅ **稳定供应保障**：
  - "常用型号常备库存"
  - "供应链稳定，不断货"
  - "订单优先级保障"

**识别技巧**：
- 公司名包含：加工、制造、定制、工厂、车间、作坊
- 业务描述：来图加工、来料加工、定制生产、OEM代工
- 地址特征：工业区、开发区、产业园
- 采购特点：询价时列出一长串产品清单，不是单一产品

**沟通要点**：
1. **首次联系**：
   - "您好，我是XXX供应链服务商，专注为加工厂提供一站式配件供应"
   - "了解到贵司主要做门窗加工，我们能提供全套五金和型材配件"
   
2. **需求挖掘**：
   - "您现在有哪些品类是固定采购的？"
   - "月用量大概多少？我们可以做专项报价"
   - "现有供应商有什么痛点？价格/质量/交期？"
   
3. **价值主张**：
   - "整合供应商，减少管理成本"
   - "统一账期，改善现金流"
   - "质量稳定，减少返工率"
   - "快速响应，不耽误生产"

**成交策略**：
- **第一步**：从1-2个常用品类切入（高频、价格敏感）
- **第二步**：建立信任后，扩展品类（捆绑销售）
- **第三步**：签订年度框架协议（锁定客户）
- **第四步**：深度合作（VMI库存管理、账期支持）

**注意事项**：
- ⚠️ 这类客户决策周期较长（1-3个月），需要多轮沟通
- ⚠️ 首单通常是小单试单，考验质量和服务
- ⚠️ 价格敏感，但更看重供应稳定性和服务响应速度
- ⚠️ 付款周期可能要求账期（30-60天），需评估风险
- ✅ 一旦建立合作，客户粘性非常高，年采购额可观

**评估要点**：
- 公司成立年限（>3年更稳定）
- 生产规模（员工数、厂房面积、设备情况）
- 主要客户类型（地产商>装修公司>散户）
- 财务状况（工商信息、付款能力）
- 行业口碑（同行评价、客户评价）

## 🎯 阶段三：智能评分系统

### 自动评分算法（0-100分）

#### 1. 信息完整度（40分）
```javascript
完整度得分 = (已填写字段总分 / 所有字段总分) × 40
```

**示例**：
- 填写了基础信息全部6项 = 30分
- 填写了公司信息3项（网站、国家、规模）= 12分
- 填写了业务信息全部4项 = 25分
- 填写了需求信息2项（需求描述、预算）= 11分
- **总分** = 78分，完整度得分 = 78% × 40 = 31.2分

---

#### 2. 客户主动性（20分）
- 提供具体需求描述 = +10分
- 提供预算范围 = +5分
- 提供时间表 = +5分

**判断标准**：
- 主动询价 > 被动响应
- 详细需求 > 简单咨询
- 明确预算 > 模糊意向

---

#### 3. 决策权限（20分）
- 是决策者（勾选） = +15分
- 职位包含"Director/Manager/VP/President/Owner" = +5分

**职位评分表**：
- CEO/President/Owner = 满分20分
- Director/VP = 18分
- Manager = 15分
- Supervisor/Coordinator = 10分
- Assistant/Specialist = 5分
- 其他 = 0分

---

#### 4. 公司规模（10分）
- 大型企业（>500人）= 10分
- 中型企业（100-500人）= 6分
- 小型企业（<100人）= 3分

---

#### 5. 采购规模（10分）
- 年采购额 >$5M = 10分
- 年采购额 $1M-$5M = 7分
- 年采购额 $500K-$1M = 4分
- 年采购额 $100K-$500K = 2分
- 年采购额 <$100K = 1分

- 紧急程度 = 高

**特征**：
- 60-89%成交概率
- 15天内预计成交
- 需重点关注

## 🎯 常见问题FAQ

### Q1: 社媒客户如何判断是否为高质量客户？
**A**: 看3个维度：
1. 主动性（主动询价 vs 被动回复）
2. 专业性（企业邮箱 vs 个人邮箱）
3. 明确性（具体需求 vs 泛泛咨询）

### Q2: 短视频引流的客户转化率低怎么办？
**A**: 优化3个环节：
1. 视频内容更精准（针对B2B采购商，非个人消费者）
2. 落地页体验优化（快速找到联系方式）
3. 首次响应速度（1小时内回复）

### Q3: 主动开发的客户如何提高回复率？
**A**: 4个技巧：
1. 邮件标题个性化（包含公司名或痛点）
2. 正文简洁（3-5句话，不超过150字）
3. 明确价值主张（解决什么问题）
4. 多渠道触达（邮件+LinkedIn+电话）

### Q4: 信息完整度不够50%是否可以投放公海池？
**A**: 不可以。必须满足最低标准：
- 公司名 ✓
- 联系人 ✓
- 邮箱 ✓
- 来源渠道 ✓
- 业务类型 ✓
- 目标行业 ✓

### Q5: 业务员认领后发现客户质量不符怎么办？
**A**: 2个选择：
1. 在7天保护期内退回公海池（需说明原因）
2. 联系运营专员，反馈客户质量问题，优化评分标准

### Q6: 本地制造加工服务商如何快速识别和开发？⭐ 新增
**A**: 识别+开发4步法：

**第1步：快速识别**
- 公司名包含：XX加工厂、XX制造、XX定制
- 询价特点：列出一长串产品清单（10+种配件）
- 地址位置：工业区、产业园、开发区
- 业务类型：来图加工、来料加工、定制生产

**第2步：需求挖掘**
- 问题1："您主要做什么类型的产品加工？"（门窗/窗帘/家具等）
- 问题2："现在有哪些配件是固定采购的？月用量大概多少？"
- 问题3："现有几家供应商？有什么痛点？"（价格/质量/交期/账期）
- 问题4："如果能整合供应商，统一账期，您感兴趣吗？"

**第3步：价值呈现**
- 一站式供应：我们产品线覆盖XX类产品80%的配件需求
- 减少管理成本：从5家供应商整合到2-3家，节省时间和人力
- 统一账期：月结30天，改善现金流
- 快速响应：常备库存，紧急补货当天发货

**第4步：分阶段成交**
- 首单试单（1-2个高频品类）→ 建立信任
- 扩展品类（3-5个品类）→ 增加份额
- 年度框架协议 → 锁定客户
- 深度合作（VMI、账期支持）→ 战略伙伴

💡 **关键提示**：这类客户决策周期较长（1-3个月），但一旦合作，年采购额$200K-$500K，客户粘性极高，值得耐心开发！
