## 🔄 数据流

### 活动管理流程
```
创建活动
  ↓
表单填写 → 验证 → 生成数据对象
  ↓
保存到 campaigns state
  ↓
保存到 localStorage('seeding_campaigns')
  ↓
UI自动更新（React state变化）
  ↓
活动列表显示新活动
```

### 活动状态更新流程
```
点击管理按钮
  ↓
打开详情Dialog（传入selectedCampaign）
  ↓
点击状态操作按钮（启动/暂停/完成）
  ↓
调用 handleUpdateCampaignStatus(id, newStatus)
  ↓
更新 campaigns state（map遍历更新）
  ↓
保存到 localStorage
  ↓
Toast提示 → UI自动更新
```

### 内容创建流程
```
点击创建内容按钮
  ↓
打开创建表单Dialog
  ↓
填写表单 → 选择类型/平台
  ↓
验证必填项
  ↓
生成 content 对象
  ↓
保存到 contents state
  ↓
保存到 localStorage('seeding_contents')
  ↓
"我的内容"列表自动显示
```

## 📊 数据统计

### 自动计算的指标
1. **进行中活动数量**: `campaigns.filter(c => c.status === 'active').length`
2. **总预算**: `campaigns.reduce((sum, c) => sum + c.budget, 0)`
3. **总曝光量**: `campaigns.reduce((sum, c) => sum + c.metrics.impressions, 0)`
4. **经销商询问数**: `campaigns.reduce((sum, c) => sum + c.metrics.dealerInquiries, 0)`
5. **预算使用率**: `(campaign.spent / campaign.budget) * 100`

## 📝 使用说明

### 创建一个完整的种草活动

1. **第一步：创建活动**
   - 点击"创建种草活动"
   - 填写活动名称："2025 Q1 北美门窗推广"
   - 选择市场：北美市场
   - 选择方式：本地代理团队
   - 设置预算：$10,000
   - 选择平台：Instagram, Facebook, YouTube
   - 填写目标："月曝光15万+，经销商询问60+"
   - 点击"创建活动"

2. **第二步：查看执行方式**
   - 切换到"执行方式"Tab
   - 点击"本地代理团队"卡片
   - 点击"查看详细流程"
   - 参考8步标准化流程

3. **第三步：创建内容**
   - 切换到"内容管理"Tab
   - 点击"创建内容"
   - 选择类型："教程内容"
   - 选择平台："YouTube"
   - 填写标题："门窗铰链安装3分钟教程"
   - 设置发布时间：2024-12-15 10:00
   - 点击"创建内容"

4. **第四步：管理活动**
   - 回到"活动概览"Tab
   - 找到创建的活动
   - 点击"管理"按钮

5. **第五步：追踪效果**
