## ❓ 常见问题

### **Q1：看不到智能推荐卡片？**

**可能原因：**
- ❌ 不是步骤1（智能推荐只在步骤1显示）
- ❌ 没有点击"选择被通知人"按钮
- ❌ 浏览器缓存问题

**解决方法：**
```
1. 确认是步骤1（标题：客户提交询价）
2. 刷新页面（Ctrl+F5 强制刷新）
3. 清除localStorage：
   localStorage.clear();
   location.reload();
4. 重新初始化数据
```

---
