## 🎯 核心功能速查

### 👁️ 预览表单
```
1. 点击表单卡片上的"预览"按钮
2. 查看完整表单结构和字段
3. 可选: 导出/复制/切换到编辑
```

### ✏️ 编辑表单
```
1. 点击表单卡片上的"编辑"按钮
2. 修改基本信息、区块、字段
3. 点击"保存更改"
```

### 📥 导出表单
```
1. 点击表单卡片上的"导出"按钮
2. 自动下载JSON配置文件
3. 文件名: {表单ID}_v{版本号}.json
```

### 📋 复制表单
```
1. 点击"复制"按钮
2. 系统创建表单副本
3. 可修改后另存为新表单
```

### 🔄 版本管理
```
1. 点击"版本"按钮展开历史
2. 创建新版本: 点击"新版本"
3. 发布版本: 点击✓图标
4. 回滚版本: 点击↻图标
```

### 📦 批量导出
```
1. 设置筛选条件（可选）
2. 点击右上角"批量导出"按钮
3. 导出所有符合条件的表单
```

## 🎯 操作流程图

### 表单预览流程
```
开始
  ↓
找到目标表单（搜索/筛选）
  ↓
点击"预览"按钮
  ↓
查看表单结构和字段
  ↓
选择操作
  ├─ 导出 → 下载JSON文件 → 结束
  ├─ 复制 → 创建副本 → 结束
  ├─ 编辑 → 进入编辑模式 → [见编辑流程]
  └─ 关闭 → 结束
```

### 表单编辑流程
```
开始
  ↓
打开编辑器（直接编辑 or 从预览切换）
  ↓
修改内容
  ├─ 基本信息（名称、类型、版本）
  ├─ 区块管理（编辑、删除、新增）
  └─ 字段管理（编辑、复制、删除、新增）
  ↓
预览效果（可选）
  ↓
保存更改
  ↓
系统提示保存成功
  ↓
结束
```

### 版本管理流程
```
创建新版本
  ↓
版本号自动+0.1
  ↓
状态设为"草稿"
  ↓
编辑新版本内容
  ↓
测试验证
  ↓
发布版本
  ↓
状态变为"已发布"
  ↓
开始使用

如需回滚:
  ↓
查看版本历史
  ↓
选择目标版本
  ↓
点击回滚
  ↓
确认操作
  ↓
恢复到指定版本
```

## 🚨 常见操作提示

### ✅ 推荐做法
- ✓ 定期备份重要表单（导出JSON）
- ✓ 创建新版本前先测试
- ✓ 使用有意义的版本说明
- ✓ 定期归档旧版本
- ✓ 保持表单结构简洁清晰

### ❌ 避免操作
- ✗ 直接删除正在使用的表单
- ✗ 跳过版本直接覆盖
- ✗ 在生产环境直接编辑
- ✗ 忽略必填字段验证
- ✗ 创建过多重复表单

### ⚠️ 注意事项
- ⚠️ 保存前预览确认
- ⚠️ 重要操作需要确认
- ⚠️ 关注成功率统计
- ⚠️ 定期检查表单性能
- ⚠️ 及时更新表单文档

## 📌 记住这些要点

1. **预览前不要编辑** - 先预览，确认后再编辑
2. **编辑前先备份** - 导出JSON作为备份
3. **保存前先预览** - 确保修改符合预期
4. **重要操作需确认** - 删除、回滚等操作会要求确认
5. **定期检查统计** - 关注使用情况和成功率
