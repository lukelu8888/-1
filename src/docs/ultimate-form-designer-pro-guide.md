# 🚀 终极表单DIY工作台 Pro - 使用指南

## 📋 概述

终极表单DIY工作台 Pro 是一个功能强大的表单设计器，支持拖拽式设计和自由调整大小功能。相比标准版，Pro版本增加了以下高级功能：

### ✨ 新增功能

#### 1. 🎯 拖拽式字段组件库
- **右侧面板**：显示所有可用的字段组件
- **可视化拖拽**：从组件库拖拽字段到画布
- **实时预览**：拖拽时显示放置位置预览

#### 2. 🔄 两种布局模式
- **网格布局模式**：传统的响应式网格布局（1/2/3列）
- **自由布局模式**：完全自由定位，支持像素级精确控制

#### 3. 📏 可调整大小功能
- **边缘拖拽**：选中字段后拖拽边缘调整宽度和高度
- **角落拖拽**：同时调整宽度和高度
- **最小尺寸限制**：防止字段过小影响可用性
- **实时预览**：调整时实时显示新尺寸

#### 4. 🎨 增强的字段定位
- **拖拽移动**：点击字段头部（带握柄图标）拖动到新位置
- **网格对齐**：在网格模式下自动对齐到网格
- **自由定位**：在自由模式下像素级精确定位
- **可视化网格**：显示背景网格辅助对齐

#### 5. 🎯 智能选择系统
- **点击选择**：点击字段或区块进行选择
- **高亮显示**：选中项显示蓝色边框
- **右侧属性面板**：实时编辑选中字段的属性
- **快速删除**：选中后显示删除按钮

## 🎮 使用方法

### 步骤1：选择模板
1. 打开终极表单DIY工作台 Pro
2. 从模板库中选择一个表单模板
3. 或点击"创建新表单"从零开始

### 步骤2：添加区块
1. 点击工具栏中的"添加区块"按钮
2. 区块将作为字段的容器
3. 可以添加多个区块组织表单结构

### 步骤3：选择布局模式
**网格模式**：
- 适合标准表单
- 自动响应式布局
- 1/2/3列可选

**自由模式**：
- 适合复杂表单
- 完全自由定位
- 支持像素级控制

### 步骤4：拖拽字段
1. 从右侧字段组件库选择字段类型
2. 拖拽到区块中
3. 在自由模式下，可以精确放置位置

### 步骤5：调整字段
**调整大小**：
1. 点击字段进行选择
2. 拖拽边缘或角落调整尺寸
3. 系统会保存新的尺寸设置

**移动位置**：
1. 点击字段头部（带握柄图标）
2. 拖拽到新位置
3. 在自由模式下可任意定位

### 步骤6：编辑属性
1. 选中字段后，右侧显示属性面板
2. 可以编辑：
   - 标签名称
   - 占位符文本
   - 必填项设置
   - 其他字段特定属性

### 步骤7：保存和导出
- **保存**：保存到数据库
- **导出JSON**：导出表单配置
- **导出PDF**：生成PDF预览
- **打印**：直接打印表单

## 🎨 快捷键

- **Ctrl/Cmd + Z**：撤销
- **Ctrl/Cmd + Y**：重做
- **Delete**：删除选中项
- **Esc**：取消选择

## 💡 最佳实践

### 1. 网格 vs 自由模式
- **简单表单**：使用网格模式，更快速
- **复杂布局**：使用自由模式，更灵活
- **混合使用**：不同区块可以使用不同模式

### 2. 字段命名
- 使用清晰的标签名称
- 添加适当的占位符提示
- 标记必填字段

### 3. 区块组织
- 按逻辑分组字段
- 使用有意义的区块标题
- 控制每个区块的字段数量

### 4. 尺寸调整
- 根据内容调整字段大小
- 保持视觉平衡
- 为长文本留足空间

### 5. 保存习惯
- 定期保存工作
- 使用撤销/重做功能
- 导出备份副本

## 🔧 技术特性

### 拖拽引擎
- **库**：react-dnd + HTML5Backend
- **支持**：字段拖拽、区块拖拽
- **预览**：实时拖拽预览

### 调整大小引擎
- **库**：re-resizable
- **方向**：8个方向调整
- **限制**：最小宽度100px，最小高度40px

### 布局系统
- **网格**：CSS Grid响应式布局
- **自由**：绝对定位布局
- **混合**：支持两种模式共存
