# 客户侧权限中心四语翻译表

## 一、说明

本文用于定义客户侧 Permission Center 的核心 UI 文案翻译。

适用范围：

- 客户侧门户
- 客户侧 Permission Center

不适用范围：

- 我方侧 ERP
- 我方侧 Permission Center

支持语言：

- English
- Español
- Português
- العربية

说明：

- 英文为默认源语言
- Arabic 页面方向为 RTL
- `Role Code`、`Module Code`、系统 code 与内部枚举不翻译

## 十一、备注

### 11.1 固定不翻译项

以下字段建议始终保留原始 code：

- `Company_Owner`
- `Company_Admin`
- `Purchase_Manager`
- `Purchaser`
- `Finance`
- `Warehouse_Receiver`
- `Viewer`

以及：

- module code
- enum code
- API 字段名
- 数据库字段名

### 11.2 语言作用域固定规则

客户侧权限中心仅跟随客户侧门户语言切换。

我方侧 ERP 不跟随客户侧 locale 变化。
