### 3. 业务类型（4种核心模式）

#### 直接采购（Trading）
- **优先级**: A级

- **利润率**: 18-25%
- **成交周期**: 30-45天
- **描述**: 客户直接采购产品，我方负责采购和发货

#### 验货服务（Inspection）
- **优先级**: A级
- **平均订单**: $20K-100K
- **利润率**: 25-35%
- **成交周期**: 15-30天
- **描述**: 为客户提供第三方验货服务

#### 代理服务（Agency）
- **优先级**: S级
- **平均订单**: $100K-1M/年
- **利润率**: 12-18%
- **成交周期**: 长期合作
- **描述**: 作为客户的中国采购代理

#### 一站式项目（Project）
- **优先级**: S级
- **平均订单**: $80K-500K
- **利润率**: 15-22%
- **成交周期**: 60-90天
- **描述**: 提供采购+验货+物流一站式服务

### 5. 意向度等级（科学分级）

#### S级（Super Hot）🔥
- **成交概率**: 90%+
- **预计成交**: 7天内
- **特征**: 高概率成交，需立即跟进
- **建议**: 最高优先级，安排高级销售

#### A级（Hot）🌡️
- **成交概率**: 60-89%
- **预计成交**: 15天内
- **特征**: 有明确采购意向，需重点关注
- **建议**: 高优先级，每天跟进

#### B级（Warm）💨
- **成交概率**: 30-59%
- **预计成交**: 30天内
- **特征**: 需重点培育，定期跟进
- **建议**: 中优先级，每周跟进

#### C级（Cold）❄️
- **成交概率**: <30%
- **预计成交**: 长期
- **特征**: 长期培育客户，低频跟进
- **建议**: 低优先级，每月跟进

### 6. 客户评分系统（0-100分）

#### 评分维度（5个维度，各20分）

| 维度 | 权重 | 评分标准 |
|------|------|----------|
| 📋 信息完整度 | 20% | 公司信息、联系方式、采购需求完整性 |
| ⚡ 沟通响应速度 | 20% | 回复速度、沟通频率、在线时长 |
| 💰 采购预算明确度 | 20% | 预算范围、采购量、目标价格明确性 |
| 👑 决策权限 | 20% | 是否决策者、审批流程、决策周期 |
| ⏰ 时间紧迫度 | 20% | 采购时间表、项目紧急程度 |

#### 评分等级

- **80-100分**: 优质客户（绿色）- 立即跟进
- **60-79分**: 良好客户（蓝色）- 重点培育
- **40-59分**: 普通客户（黄色）- 定期跟进
- **0-39分**: 低质客户（红色）- 低频维护

### 8. 背调状态管理

| 状态 | 说明 | 后续动作 |
|------|------|----------|
| ⏱️ 待背调 | 客户刚进入公海池 | 业务员认领后启动背调 |
| 🔄 背调中 | 正在进行背景调查 | 等待背调结果 |
| ✅ 背调通过 | 客户资质良好 | 可以正式开发 |
| ⚠️ 背调存疑 | 发现可疑信息 | 需要进一步核实 |
| ❌ 背调失败 | 客户资质不符 | 不建议合作 |

## 🔄 业务流程

### 1. 客户投放流程（运营专员）
```
社媒获客 → 初步筛选 → 信息完善 → 评分打分 → 投放到公海池 → 监控认领情况
```

### 2. 客户认领流程（业务员）
```
浏览公海池 → 筛选目标客户 → 填写认领原因 → 制定跟进计划 → 确认认领 → 进入7天保护期
```

### 3. 背调流程
```
认领客户 → 启动背调 → 工商信息查询 → 信用评估 → 行业口碑调查 → 形成背调报告 → 决定是否继续
```

### 4. 保护期机制
- **时长**: 7天
- **要求**: 
  - 完成客户背调
  - 至少3次跟进记录
  - 更新客户信息
- **违规**: 未完成要求，客户退回公海池
- **成功**: 客户转入私海池，正式开发

## 💡 使用建议

### 运营专员最佳实践
1. 每日投放客户前，确保信息完整度达到80%以上
2. 根据业务员专长分配区域和行业
3. 定期清理低质量客户（评分<40分）
4. 关注认领率，优化客户质量

### 业务员最佳实践
1. 优先认领S级和A级客户
2. 认领前查看完整客户资料和评分明细
3. 制定详细的7天跟进计划
4. 及时完成背调和跟进记录
5. 合理分配每月10个认领名额

### 区域主管最佳实践
1. 监控公海池健康度指标
2. 根据业务员能力调整分配规则
3. 及时处理保护期违规情况
4. 定期分析转化率，优化流程
