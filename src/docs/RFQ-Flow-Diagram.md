## 🎯 关键函数调用链

```
用户点击"提交"
    │
    ├─▶ handleSubmitRFQToSupplier(rfq)
    │       │
    │       ├─▶ 检查重复: existingSupplierRFQs.some(...)
    │       │
    │       ├─▶ 更新状态: updateRFQ(rfq.id, {status: 'sent'})
    │       │       │
    │       │       └─▶ RFQContext.updateRFQ()
    │       │               │
    │       │               └─▶ setRFQs([...updated])
    │       │                       │
    │       │                       └─▶ localStorage.setItem('rfqs', ...)
    │       │
    │       ├─▶ 构建数据: supplierRFQData = {...}
    │       │
    │       ├─▶ 保存数据: localStorage.setItem('supplierRFQs', ...)
    │       │
    │       ├─▶ 触发事件: window.dispatchEvent(new Event('storage'))
    │       │
    │       └─▶ 显示提示: toast.success(...)
    │
    └─▶ Supplier端自动接收
            │
            ├─▶ RFQContext监听storage事件
            │       │
            │       └─▶ handleStorageChange()
            │               │
            │               └─▶ 重新加载supplierRFQs
            │                       │
            │                       └─▶ 合并到rfqs状态
            │
            └─▶ getRFQsBySupplier(user.email)
                    │
                    └─▶ 过滤出当前供应商的RFQ
                            │
                            └─▶ categorizedRFQs.pending
                                    │
                                    └─▶ 显示在客户需求池
```

## 🔔 通知机制

```
Admin点击提交
    ↓
触发storage事件
    ↓
window.dispatchEvent(new Event('storage'))
    ↓
Supplier端RFQContext监听
    ↓
React.useEffect(() => {
  const handleStorageChange = () => {
    // 重新加载supplierRFQs
    // 更新rfqs状态
  };
  
  window.addEventListener('storage', handleStorageChange);
  return () => window.removeEventListener('storage', handleStorageChange);
}, []);
    ↓
Supplier端自动刷新列表
    ↓
显示新的询价单
```

```json
{
  "id": "rfq-1234567890",
  "rfqNumber": "XJ-251221-3862",
  "supplierRfqNo": "XJ-251221-3862",
  "supplierCode": "DG-HX-001",
  "supplierEmail": "supplier1@example.com",
  "sourceQRNumber": "QR-251221-0001",
  "status": "pending",
  "sentDate": "2025-12-21",
  "products": [...],
  "documentData": {
    ...

  },
  "buyerContact": "采购部",
  "buyerEmail": "purchasing@gosundafu.com"

```

**关键差异**:
- `status`: Admin端为`sent`，Supplier端为`pending`
- `inquiryReference`: 只包含客户要求部分，不含内部说明
