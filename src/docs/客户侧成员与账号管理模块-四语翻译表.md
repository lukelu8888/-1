# 客户侧成员与账号管理模块四语翻译表

## 一、说明

本文用于定义客户侧 `People & Accounts Center` 的核心 UI 文案翻译。

支持语言：

- English
- Español
- Português
- العربية

说明：

- 英文为默认源语言
- Arabic 页面方向为 RTL
- role code 与系统 code 不翻译

## 九、备注

### 固定不翻译项

以下建议始终保留 code：

- `Company_Owner`
- `Company_Admin`
- `Purchase_Manager`
- `Purchaser`
- `Finance`
- `Warehouse_Receiver`
- `Viewer`

### 语言作用域

本模块只跟随客户侧主页语言切换。  
我方侧 ERP 不跟随客户侧 locale 变化。
