## 📊 一、数据录入模块

### 1.1 历史销售数据

**录入字段：**
- 去年销售额（USD）
- 两年前销售额（USD）
- 三年前销售额（USD）

**计算逻辑：**
```javascript
// 年均增长率
avgGrowth = [(去年-两年前)/两年前 + (两年前-三年前)/三年前] / 2

// 数据稳定性（变异系数）
CV = 标准差 / 平均值 × 100%

// 支撑度评分
支撑度 = 70分（基础） 
       + 增长率加分（0-20分）
       + 稳定性加分（0-10分）
```

**评分标准：**
- 增长率 >10%: +20分
- 增长率 5-10%: +15分
- 增长率 0-5%: +10分
- CV <20%（高稳定）: +10分
- CV 20-40%（中稳定）: +5分

**结果示例：**
> 年均增长12.5%，数据稳定性优秀 → 支撑度95%

---

### 1.2 市场环境数据

**录入字段：**
- 市场总规模（USD）
- 当前市场份额（%）
- 目标市场份额（%）
- 行业年增长率（%）
- 竞争对手平均增长率（%）

**计算逻辑：**
```javascript
// 市场增长空间
growthSpace = (目标份额 - 当前份额) / 当前份额 × 100%

// 机会度评分
机会度 = 60分（基础）
       + 行业增长加分（0-15分）
       + 市场空间加分（0-15分）
       + 竞争优势加分（0-10分）
```

**评分标准：**
- 行业增长 >12%: +15分
- 行业增长 8-12%: +10分
- 行业增长 4-8%: +5分
- 市场空间 >15%: +15分
- 市场空间 10-15%: +10分
- 市场空间 5-10%: +5分
- 我们目标增长 > 竞争对手: +10分

**结果示例：**
> 行业增长14.2%，市场空间8.2%，具备良好增长机会 → 机会度88%

---

### 1.3 团队资源数据

**录入字段：**
- 销售团队人数
- 人均年成单数
- 平均订单金额（USD）
- 线索转化率（%）
- 当前销售周期（天）
- 目标销售周期（天）

**计算逻辑：**
```javascript
// 年产能
annualCapacity = 团队人数 × 人均成单数 × 平均订单额

// 流程效率
efficiency = 1 - |当前周期 - 目标周期| / 当前周期

// 执行力评分
执行力 = 65分（基础）
       + 转化率加分（0-15分）
       + 人均产能加分（0-10分）
       + 流程效率加分（0-10分）
```

**评分标准：**
- 转化率 >30%: +15分
- 转化率 25-30%: +12分
- 转化率 20-25%: +8分
- 转化率 15-20%: +5分
- 人均成单 >25单: +10分
- 人均成单 20-25单: +7分
- 人均成单 15-20单: +4分
- 流程效率评分: efficiency × 10分

**结果示例：**
> 团队年产能$7.4M，转化率28%，执行能力强 → 执行力92%

### 1.4 战略规划数据

**录入字段：**
- 新产品线数量 + 预期收入
- 新区域市场数量 + 预期收入
- 重点大客户数量 + 平均年收入
- 战略合作伙伴数量 + 合作预期收入

**计算逻辑：**
```javascript
// 战略增量收入
totalRevenue = 新产品收入 + 新区域收入 
             + (大客户数量 × 客户平均收入) 
             + 合作收入

// 增长性评分
增长性 = 60分（基础）
       + 新产品加分（0-15分）
       + 新区域加分（0-15分）
       + 大客户加分（0-15分）
       + 合作加分（0-5分）
```

**评分标准：**
- 有新产品: +10分，单品贡献>$500K: 额外+5分
- 有新区域: +10分，预期收入>$500K: 额外+5分
- 大客户>5个: +10分，平均单价>$200K: 额外+5分
- 有战略合作: +5分

**结果示例：**
> 战略举措预计贡献$2.8M增量收入 → 增长性90%

## 🧮 二、核心计算引擎

### 2.1 综合可达成度算法

**加权公式：**
```javascript
综合可达成度 = 历史支撑度 × 25%
           + 市场机会度 × 25%
           + 团队执行力 × 30%
           + 战略增长性 × 20%
```

**权重说明：**
- **历史支撑 25%**: 基于过往业绩的可靠性
- **市场机会 25%**: 外部环境的有利程度
- **团队执行 30%**: 内部资源的承载能力（最重要）
- **战略增长 20%**: 新增长点的贡献潜力

**评估标准：**
- ≥85分: ✅ 目标设定合理，可达成性高
- 70-84分: ⚠️ 目标略有挑战，需强化执行
- <70分: ❌ 目标设定偏高，建议调整

**实例计算：**
```
历史支撑95% × 0.25 = 23.75
市场机会88% × 0.25 = 22.00
团队执行92% × 0.30 = 27.60
战略增长90% × 0.20 = 18.00
────────────────────────
综合可达成度 = 91.35% ≈ 91%
```
> ✅ 年度目标$18.5M具有91%的可实现性

### 2.2 AI预测算法

**预测公式：**
```javascript
预测值 = 基准值 × (1 + 增长率 × 月数/12) × 季节性因子

其中：
基础增长率 = (历史平均增长率 + 市场增长空间) / 2
季节性因子 = 根据历史数据统计的各月销售权重
```

**置信区间：**
```javascript
保守预测（95%置信）:
  predicted × 0.95, 区间 ±5%

正常预测（90%置信）:
  predicted × 1.0, 区间 ±8%

乐观预测（85%置信）:
  predicted × 1.05, 区间 ±10%
```

**季节性因子示例：**
```javascript
{
  1月: 0.95  // 新年淡季
  2月: 0.85  // 春节假期
  3月: 1.05  // 复工高峰
  ...
  11月: 1.12 // 备货旺季
  12月: 1.18 // 年底冲刺
}
```

**预测示例（12月）：**
```
基准值: $1,420,000（11月实际）
基础增长率: (12.5% + 8.2%) / 2 = 10.35%
季节性因子: 1.18
预测天数: 1个月

预测值 = 1,420,000 × (1 + 0.1035 × 1/12) × 1.18
      = 1,420,000 × 1.00863 × 1.18
      = $1,690,500

置信区间（90%）:
  最小值: $1,555,260 (-8%)
  最大值: $1,825,740 (+8%)
  置信度: 90%
```

## 📈 三、目标管理与执行

### 3.1 目标设定SMART原则

**S - Specific（具体明确）**
- ✅ 年度目标: $18,500,000
- ✅ 季度目标: Q4 $5,500,000
- ✅ 月度目标: 11月 $1,750,000
- ❌ 避免: "提升销售业绩"

**M - Measurable（可衡量）**
- ✅ 完成率、同比增长、市场份额
- ✅ 新客户数、客单价、转化率
- ❌ 避免: "做得更好"

**A - Achievable（可达成）**
- ✅ 综合可达成度91%（基于四维评估）
- ✅ 历史增长12.5% vs 目标增长15.8%（合理挑战）
- ❌ 避免: 超出资源承载能力

**R - Relevant（相关性）**
- ✅ 与公司战略一致
- ✅ 与市场机会匹配
- ✅ 与团队能力匹配

**T - Time-bound（有时限）**
- ✅ 年度 → 季度 → 月度 → 周度 → 日度
- ✅ 每个层级都有明确截止日期

## 📚 七、常见问题FAQ

### Q1: 为什么我的可达成度只有68%？
**A**: 可能原因：
1. 历史增长率过低（<5%）
2. 市场空间不足（<5%）
3. 团队转化率偏低（<20%）
4. 缺乏战略增长点

**建议**: 调整目标至合理区间，或强化短板指标

---

### Q2: AI预测值和实际差异很大怎么办？
**A**: 预测基于：
1. 历史趋势（过去3年数据）
2. 市场增长率（行业平均）
3. 季节性因素（月度权重）

**建议**: 
- 录入更准确的历史数据
- 更新市场增长率
- 调整季节性因子

---

### Q3: 如何设置合理的销售周期？
**A**: 参考标准：
- B2B简单产品: 15-30天
- B2B复杂产品: 30-90天
- B2B企业级: 90-180天

**目标周期** = 当前周期 × 0.8-0.9（提效空间）

---

### Q4: 执行策略的贡献率如何计算？
**A**: 
```
贡献率 = 该策略预期收入 / 总增量收入 × 100%

示例：
新客户开发预期$450K / 总增量$1.8M = 25%
```
