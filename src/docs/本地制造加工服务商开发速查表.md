## 🎯 4步成交法（记住这个顺序）

### Step 1: 切入品类（1周内）
**目标**：从1-2个高频、价格敏感的品类切入

**话术**：
```
"我看您每月XX（螺丝/密封条）的用量不小，
我们这个价格比市场低8-10%，质量稳定。
要不先试一单看看？"
```

**推荐切入品类**：
- 门窗厂：螺丝、密封条（高频低价）
- 窗帘厂：滑轮、挂钩（消耗品）
- 家具厂：铰链、滑轨（标准件）

---

### Step 2: 建立信任（2-4周）
**目标**：首单试单，证明质量和服务

**关键动作**：
- ✅ 快速发货（3天内）
- ✅ 质量OK（减少退换货）
- ✅ 主动跟进（安装使用反馈）
- ✅ 小问题快速解决

**话术**：
```
"这批货用得怎么样？
还有哪些配件需要采购的，我一起给您报个价？"
```

---

### Step 3: 扩展品类（1-2个月）
**目标**：从2个品类扩展到5-8个品类

**策略**：
- 捆绑销售（"这几个一起采购，我给您更优惠的价格"）
- 专项套餐（"门窗配件套餐，一次性备齐"）
- 新品推荐（"这个新款XX很多客户用，要不试试？"）

**目标占比**：
- 首单：客户采购额的5-10%
- 第2月：20-30%
- 第3月：40-50%
- 第4月：成为主要供应商（60%+）

---

### Step 4: 锁定客户（3个月后）
**目标**：签订年度框架协议，成为战略供应商

**协议内容**：
- 年度采购额承诺（例如$200K）
- 统一账期（月结30天）
- 价格锁定（季度调整）
- 优先供货（旺季不断货）
- VMI库存管理（可选）

**话术**：
```
"咱们合作这几个月，相互都比较了解了。
我们公司有一个年度框架协议，
签了之后价格更优惠，账期也更灵活，您看这个方案？"
```

## ⚠️ 7个常见错误（避免踩坑）

| ❌ 错误做法 | ✅ 正确做法 | 原因 |
|-----------|-----------|------|
| 一次性报全部品类价格 | 先报1-2个核心品类 | 避免信息过载，客户懒得看 |
| 强调产品质量好 | 强调一站式供应、减少供应商 | 客户更关心采购效率 |
| 要求大额首单 | 小单试单，快速交付 | 建立信任比首单金额重要 |
| 只谈价格 | 谈价值（统一账期、快速响应） | 价格战没前途 |
| 客户不回复就放弃 | 定期跟进（每周1次） | 决策周期长，需要持续触达 |
| 只联系采购 | 同时联系老板/生产经理 | 找到决策者和影响者 |
| 承诺做不到的账期 | 实事求是，账期逐步开放 | 违约会失去信任 |

### 案例：XX门窗加工厂（记住这个故事，方便给新客户讲）

**关键成功因素**：
- ✅ 快速交付（3天到货）
- ✅ 质量稳定（退货率<1%）
- ✅ 价格合理（比市场低8%）
