## 🐛 常见问题排查

### **问题1：销售总监仍然能看到供应商管理**

**原因：** AdminDashboard.tsx中菜单渲染逻辑问题

**检查：**
```typescript
// AdminDashboard.tsx
const hasSupplierAccess = currentUser 
  ? hasPermission(currentUser, 'access:supplier_management' as Permission)
  : false;

// 供应链管理分组应该基于hasSupplierAccess显示
{hasSupplierAccess && (
  <li>
    {/* 供应商管理菜单 */}
  </li>
)}
```

### **问题2：区域主管看到其他区域数据**

**原因：** 数据过滤逻辑未正确实施

**检查：**
```typescript
// 数据查询时应该过滤
const filteredCustomers = currentUser.region === 'all'
  ? customers  // 销售总监看所有
  : customers.filter(c => c.region === currentUser.region); // 区域主管只看本区域
```

### **问题3：财务模块显示不正确**

**原因：** 菜单项没有基于权限隐藏

**检查：**
```typescript
// 应收账款菜单
{hasPermission(currentUser, 'access:receivables') && (
  <li>应收账款</li>
)}

// 应付账款菜单
{hasPermission(currentUser, 'access:payables') && (
  <li>应付账款</li>
)}
```
