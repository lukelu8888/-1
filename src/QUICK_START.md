# ⚡ 5分钟快速启动指南

> **目标**：让您在5分钟内看到运行中的网站

---

## 📋 **准备工作（仅第一次需要）**

### **步骤1：安装Node.js**

**检查是否已安装**：
1. 按 `Win + R`
2. 输入 `cmd`，按回车
3. 在黑色窗口输入：`node -v`
4. 如果显示版本号（如 `v18.17.0`），说明已安装 ✅
5. 如果提示"不是内部或外部命令"，继续下面的安装步骤 ❌

**安装步骤**：
1. 访问：https://nodejs.org/zh-cn/
2. 点击绿色按钮下载"长期支持版"
3. 下载完成后，双击安装包
4. 一路点"下一步"（保持默认设置）
5. 安装完成后**重启电脑**
6. 重新检查（运行 `node -v`）

---

## 🚀 **启动网站（超简单）**

### **方法A：使用启动脚本（推荐）⭐⭐⭐**

1. **找到项目文件夹**
   - 就是包含这个文件的文件夹

2. **双击运行**
   - 找到文件：`启动网站.bat`
   - 双击它

3. **等待启动**
   - 会看到黑色命令窗口
   - 自动检查环境、安装依赖、启动网站
   - 第一次运行会比较慢（需要下载依赖）

4. **看到成功提示**
   ```
   ====================================
      正在启动网站...
   ====================================
   
   启动成功后，浏览器访问：
   
      👉 http://localhost:5173
   
   ====================================
   
   VITE v5.x.x  ready in xxx ms
   
   ➜  Local:   http://localhost:5173/
   ```

5. **打开浏览器**
   - 在地址栏输入：`http://localhost:5173`
   - 看到网站首页 🎉

---

### **方法B：手动命令（备用）**

1. **打开命令提示符**
   - 按 `Win + R`
   - 输入 `cmd`
   - 按回车

2. **进入项目目录**
   ```
   cd 您的项目路径
   ```
   **技巧**：直接把项目文件夹拖到命令窗口

3. **安装依赖（仅第一次）**
   ```
   npm install
   ```
   等待完成...

4. **启动网站**
   ```
   npm run dev
   ```

5. **打开浏览器**
   - 访问：`http://localhost:5173`

## 🎯 **启动后如何访问业务流程编辑器**

1. **点击右上角**
   - 找到红色的 **"Admin Portal"** 按钮
   - 点击

2. **登录**
   - 输入管理员账号密码
   - 如果没有账号，点击"注册"

3. **找到流程编辑器**
   - 登录后，左侧菜单往下滚动
   - 找到 **"业务流程编辑器"** (有NEW标签)
   - 点击进入

4. **导出流程文档**
   - 点击右上角绿色按钮 **"导出易读文档"**
   - 用记事本打开查看

## ❌ **故障排除**

### **问题1：双击 `启动网站.bat` 后闪退**
**原因**：Node.js未安装
**解决**：
1. 安装Node.js（见上面"准备工作"）
2. 重启电脑
3. 再次双击 `启动网站.bat`

---

### **问题2：卡在"正在安装依赖"很久**
**原因**：网络下载慢
**解决**：
1. 等待完成（第一次安装需要5-10分钟）
2. 或者按 `Ctrl + C` 停止
3. 在命令窗口输入：
   ```
   npm config set registry https://registry.npmmirror.com
   npm install
   ```
4. 使用国内镜像源，速度更快

---

### **问题3：浏览器访问localhost:5173显示"无法访问"**
**原因**：网站未启动或端口被占用
**解决**：
1. 检查命令窗口是否显示 `Local: http://localhost:5173/`
2. 如果没有，检查是否有错误信息
3. 尝试关闭命令窗口，重新运行 `启动网站.bat`

---

### **问题4：网站启动了，但页面空白**
**解决**：
1. 按 `F12` 打开浏览器开发者工具
2. 点击"Console"标签，查看错误
3. 按 `Ctrl + F5` 强制刷新页面
4. 或清除浏览器缓存后重试

---

## 🔄 **如何停止网站**

### **方法1：**
- 在命令窗口按 `Ctrl + C`
- 提示"终止批处理操作吗"，输入 `Y`，按回车

### **方法2：**
- 直接关闭命令窗口

**注意**：停止后，`http://localhost:5173` 就无法访问了，需要重新启动

## 💡 **小技巧**

### **创建桌面快捷方式**
1. 右键拖动 `启动网站.bat` 到桌面
2. 选择"在当前位置创建快捷方式"
3. 重命名为"启动外贸系统"
4. 以后双击桌面图标即可启动

### **让浏览器记住网址**
1. 第一次访问 `http://localhost:5173` 后
2. 按 `Ctrl + D` 添加到书签
3. 重命名为"高盛达富外贸系统"
4. 以后点击书签即可

### **同时打开多个窗口**
1. 启动网站后
2. 可以同时打开：
   - 客户Portal：`http://localhost:5173`
   - Admin Portal：点击右上角登录
   - 在不同浏览器窗口查看不同角色视图
