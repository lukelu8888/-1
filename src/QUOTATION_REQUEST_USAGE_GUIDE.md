# 📋 向采购员请求报价 - 完整使用指南

## ✅ 功能已完成

业务员收到客户询价单后，可以向采购员发起报价请求的完整功能已实现！

---

## 🎯 业务流程

```
📧 客户提交询价单 (Customer Inquiry)
  ↓
👤 业务员登录Admin Portal
  ↓
📋 业务员在"询价管理"中查看客户询价
  ↓
👁️ 业务员点击"查看"打开询价详情
  ↓
📝 业务员点击"向采购员请求报价"按钮
  ↓
📅 填写期望报价日期和备注
  ↓
✅ 提交报价请求 (Quotation Request)
  ↓
👨‍💼 采购员收到报价请求
  ↓
🔍 采购员向多个供应商询价 (RFQ)
```

---

## 📍 如何使用

### 步骤1：业务员登录
1. 访问Admin Portal登录页面
2. 使用业务员账号登录（如：`sales@cosun.com`）

### 步骤2：查看客户询价
1. 进入 **询价管理** 模块
2. 可以看到所有客户提交的询价单
3. 每条询价单显示：
   - 询价单号（如：RFQ-NA-251217-1234）
   - 客户名称
   - 产品信息
   - 状态（待处理/已报价等）

### 步骤3：打开询价详情
1. 点击询价单号或"查看"按钮
2. 弹出询价详情对话框
3. 显示完整的客户询价信息：
   - 公司信息
   - 联系人信息
   - 产品清单
   - 数量、规格等

### 步骤4：向采购员请求报价
1. 在询价详情底部的"管理操作"区域
2. 找到 **"向采购员请求报价"** 按钮（蓝色）
3. 点击按钮，弹出报价请求对话框

### 步骤5：填写报价请求信息
对话框中会显示：

#### ✅ 自动填充的信息：
- **客户询价单号**：RFQ-NA-251217-1234
- **客户名称**：Home Depot Inc.
- **产品清单**：所有询价的产品
- **数量信息**：每个产品的数量

#### ✍️ 需要填写的信息：
- **期望报价日期** <span style="color:red">*必填</span>
  - 选择采购员需要提供报价的截止日期
  - 例如：2025-12-31
  
- **备注信息**（可选）
  - 给采购员的特殊说明
  - 例如：
    - "客户很急，请尽快"
    - "这是VIP客户，价格要有竞争力"
    - "需要提供FOB和CIF两种报价"

### 步骤6：提交报价请求
1. 确认信息无误
2. 点击 **"提交报价请求"** 按钮
3. 系统自动生成报价请求单号（QR-NA-251217-XXXX）
4. 显示成功提示

## 🎯 状态流转

报价请求的状态流转：

```
pending (待处理)
  ↓ 采购员开始询价
processing (处理中)
  ↓ 采购员收到供应商报价
quoted (已报价)
  ↓ 业务员收到成本报价
completed (已完成)
```

或者：
```
pending (待处理)
  ↓ 业务员取消
cancelled (已取消)
```

## 🐛 常见问题

### Q1: 点击按钮没有反应？
**A**: 检查是否选中了询价单，确保在询价详情对话框中操作。

### Q2: 提示"请设置期望报价日期"？
**A**: 期望报价日期是必填项，必须选择一个日期。

### Q3: 看不到"向采购员请求报价"按钮？
**A**: 确保你是以业务员身份登录，并且在询价详情页面的"管理操作"区域查找。

### Q4: 提交后如何查看报价请求？
**A**: 目前数据保存在localStorage中，可以在Console中查看：
```javascript
JSON.parse(localStorage.getItem('quotationRequests'))
```
