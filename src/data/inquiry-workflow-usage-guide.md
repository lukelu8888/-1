# 🚀 询价到合同流程使用指南

## 快速开始

### 方式一：直接加载预设流程（推荐）⭐

1. **打开流程编辑器**
   ```
   导航到: 工作台 → 业务流程编辑器 Pro
   或直接访问: /workflow-editor-pro
   ```

2. **点击"加载询价流程"按钮**
   - 位于页面右上角，紫色按钮
   - 图标：📋 加载询价流程

3. **确认加载**
   - 系统会提示：🔄 确定要加载"客户询价到合同流程"吗？
   - 点击"确定"

4. **流程加载成功**
   - ✅ 已加载"询价到合同"流程！共5个阶段，21个步骤
   - 流程自动显示在编辑器中

## 📝 流程操作指南

### 查看阶段

1. **左侧阶段列表**
   - 显示所有5个阶段
   - 点击阶段卡片查看详情
   - 阶段名称、描述、步骤数量

2. **添加新阶段**
   - 点击"添加阶段"按钮
   - 填写阶段信息
   - 保存

### 编辑步骤

1. **选择阶段**
   - 在左侧点击阶段

2. **选择步骤**
   - 在中间列表点击步骤卡片
   - 步骤高亮显示（蓝色背景）

3. **编辑步骤信息**
   - 右侧显示步骤详情
   - 点击"编辑"按钮
   - 修改以下内容：
     - ✏️ 步骤ID
     - 📝 操作名称
     - 📄 描述
     - 👤 执行角色
     - 📋 必需字段
     - 🔔 触发器
     - → 下一步骤

4. **保存修改**
   - 点击"保存更改"
   - ✅ 步骤已更新

### 添加步骤

1. **选择阶段**
2. **点击"添加步骤"**
3. **填写步骤信息**
   - 步骤ID（唯一标识）
   - 操作名称
   - 描述
   - 执行角色
   - 必需字段列表
4. **保存**

### 删除步骤

1. **选择步骤**
2. **点击"删除"按钮**
3. **确认删除**
   - ⚠️ 确定要删除此步骤吗？

### 调整步骤顺序（V5.0新功能）✨

**重要说明**: V5.0版本已将拖拽改为按钮移动方式！

1. **选择要移动的步骤**
   - 点击步骤卡片
   - 步骤高亮，显示移动按钮

2. **使用移动按钮**
   - **↑ 向上移动**: 将步骤向上移动一位
   - **↓ 向下移动**: 将步骤向下移动一位

3. **移动限制**
   - 第一个步骤：向上按钮禁用
   - 最后一个步骤：向下按钮禁用

4. **移动成功**
   - ✅ 步骤已移动
   - 步骤顺序立即更新

**优势**:
- ✅ 操作更精确
- ✅ 移动设备友好
- ✅ 无需拖拽技巧
- ✅ 代码减少53%

## 🔍 常见问题

### Q1: 如何修改金额阈值？

**A**: 
1. 选择"金额自动路由"步骤
2. 点击"编辑"
3. 修改触发器中的条件：
   ```javascript
   total_amount < 20000  // 修改为你的阈值
   total_amount >= 200000 // 修改为你的阈值
   ```

### Q2: 如何添加新的审批角色？

**A**:
1. 在步骤编辑器中选择"执行角色"
2. 输入新角色ID（如：ceo）
3. 系统会自动添加该角色
4. 保存步骤

### Q3: 步骤顺序调整后如何保存？

**A**:
- 步骤移动后会自动保存
- 点击"保存配置"确保永久保存
- 导出JSON文件作为备份

### Q4: 如何删除整个阶段？

**A**:
1. 选择阶段
2. 点击阶段卡片上的删除按钮
3. 确认删除
4. ⚠️ 注意：阶段下的所有步骤也会被删除

### Q5: 能否添加更多阶段？

**A**:
- 完全可以！
- 点击"添加阶段"按钮
- 填写阶段信息
- 添加步骤到新阶段

### Q6: 如何设置条件分支？

**A**:
1. 编辑步骤
2. 在"触发器"部分添加分支
3. 设置条件表达式
4. 指定不同的next_steps

### Q7: 移动按钮点击没反应？

**A**:
- 确保步骤已被选中（蓝色高亮）
- 检查是否是第一个/最后一个步骤
- 查看控制台是否有错误
- 尝试刷新页面

### Q8: 导入JSON失败？

**A**:
- 检查JSON格式是否正确
- 确保是从本系统导出的文件
- 查看控制台错误信息
- 尝试重新下载模板文件
