# 🎯 向采购员请求报价弹窗 - 台湾大厂风格重设计

## ✅ 重设计完成

已按照**台湾大厂风格**重新设计"向采购员请求报价"弹窗：专业化、紧凑化、高信息密度。

---

## 🎨 设计理念

### 核心原则
1. **信息聚焦** - 只显示产品相关信息
2. **表格化展示** - 清晰、专业、易读
3. **紧凑布局** - 高信息密度
4. **橙色强调** - 重点信息突出

### 台湾大厂风格特征
- ✅ 表格化（不是卡片）
- ✅ 边框清晰
- ✅ 灰色背景区分表头
- ✅ 紧凑的间距
- ✅ 专业的字体和排版
- ✅ 橙色高亮关键数据

## 📊 新界面结构详解

### 1. 标题栏（简洁）
```
┌──────────────────────────────────────────────────┐
│ 📤 向采购员请求报价              [RFQ-EA-251217-0004] │
│ 采购员将基于产品需求向采购询价并提供成本报价        │
└──────────────────────────────────────────────────┘
```
- 左侧：标题 + 图标
- 右侧：询价单号（Badge）
- 下方：一句话说明

### 2. 产品清单表格（核心）
```
┌─────────────────────────────────────────────────────────────┐
│ # │ 产品名称      │ 型号/编号    │ 规格说明    │ 需求数量  │ 特殊要求 │
├───┼──────────────┼─────────────┼────────────┼──────────┼─────────┤
│ ① │ LED Light... │ LED-60W-A19 │ 60W A19... │ ┌─────┐ │ UL cert │
│   │ [Electrical] │             │            │ │100  │ │         │
│   │              │             │            │ │pcs  │ │         │
│   │              │             │            │ └─────┘ │         │
└───┴──────────────┴─────────────┴────────────┴──────────┴─────────┘
```

#### 列宽分配：
| 列 | 宽度 | 说明 |
|----|------|------|
| # | 48px | 序号 |
| 产品名称 | 200px | 名称+类别 |
| 型号/编号 | 120px | 等宽字体 |
| 规格说明 | 250px | 详细描述 |
| **需求数量** | **120px** | **橙色高亮** |
| 特殊要求 | 200px | 黄色背景 |

#### 表头样式：
- 背景：`bg-slate-100`
- 字体：`font-semibold text-slate-700`
- 需求数量列：`text-orange-700`（橙色强调）

#### 表格行：
- 偶数行：白色
- Hover：`hover:bg-orange-50/50`
- 行高：`py-3`（紧凑）

#### 需求数量单元格（重点）：
```
┌──────────────┐
│ 📦 100 pcs   │ ← 橙色边框+背景
└──────────────┘
```
- 背景：`bg-orange-100`
- 边框：`border-orange-300`
- 图标：📦
- 数字：**粗体大号**
- 单位：小号

### 3. 日期和备注（横向布局）
```
┌────────────────────────┬────────────────────────┐
│ 📅 期望报价日期 *      │ 📝 给采购员的备注      │
│ [日期选择器]           │ [文本框]               │
│ 说明文字...            │ ⚠️ 隐私提示...         │
└────────────────────────┴────────────────────────┘
```
- 左右各占50%
- 紧凑的2列布局

### 4. 底部操作栏
```
┌──────────────────────────────────────────────────┐
│ 共 3 个产品，总计 300 件        [取消] [📤 提交] │
└──────────────────────────────────────────────────┘
```
- 左侧：统计信息
- 右侧：操作按钮

## 🎯 台湾大厂风格特征

### ✅ 已应用

#### 1. 表格化布局
```
不使用卡片 → 使用表格
原因：专业、紧凑、信息密度高
```

#### 2. 清晰的边框
```
表格：border rounded-lg
表头：border-b
单元格：py-3 px-4
```

#### 3. 灰色背景区分
```
表头：bg-slate-100
Hover：bg-orange-50/50
```

#### 4. 紧凑的间距
```
行高：py-3（不是 py-4）
列间距：px-4
表格间距：space-y-3（不是 space-y-6）
```

#### 5. 专业的字体
```
标题：text-xl font-bold
正文：text-sm
型号：font-mono（等宽字体）
数字：font-bold text-base
```

#### 6. 橙色强调
```
需求数量：
- bg-orange-100
- border-orange-300
- text-orange-900
```
