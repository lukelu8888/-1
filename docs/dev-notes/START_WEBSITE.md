# 🚀 如何启动网站 - 超简单版

## ⚠️ **前置要求**

在启动网站之前，您需要确保安装了：

1. **Node.js** (建议版本 18 或以上)
   - 下载地址：https://nodejs.org/
   - 下载"LTS"版本（长期支持版）

2. **代码编辑器** (可选，推荐 VS Code)
   - 下载地址：https://code.visualstudio.com/

---

## 📝 **启动步骤**

### **方法1：Windows系统（命令提示符）**

1. **打开命令提示符**
   - 按键盘上的 `Win + R`
   - 输入 `cmd`
   - 按回车

2. **进入项目文件夹**
   ```
   cd 您的项目路径
   ```
   例如：
   ```
   cd C:\Users\YourName\Desktop\cosun-project
   ```

3. **安装依赖（第一次需要）**
   ```
   npm install
   ```
   **注意**：这一步只需要做一次，以后不用再做

4. **启动网站**
   ```
   npm run dev
   ```

5. **等待启动**
   - 会看到类似这样的信息：
   ```
   VITE v5.x.x  ready in xxx ms

   ➜  Local:   http://localhost:5173/
   ➜  Network: http://192.168.x.x:5173/
   ```

6. **打开浏览器**
   - 在浏览器地址栏输入：`http://localhost:5173`
   - 按回车

---

### **方法2：使用VS Code（推荐）⭐**

1. **打开VS Code**

2. **打开项目文件夹**
   - 点击"文件" → "打开文件夹"
   - 选择您的项目文件夹

3. **打开终端**
   - 点击顶部菜单：`终端` → `新建终端`
   - 或按快捷键：`` Ctrl + ` ``（反引号键）

4. **安装依赖（第一次需要）**
   在终端输入：
   ```
   npm install
   ```

5. **启动网站**
   在终端输入：
   ```
   npm run dev
   ```

6. **打开浏览器**
   - 终端会显示：`Local: http://localhost:5173/`
   - 按住 `Ctrl` 键，点击这个链接
   - 或手动在浏览器输入：`http://localhost:5173`

---

## ✅ **启动成功的标志**

当您看到以下信息时，说明启动成功了：

```
  VITE v5.4.10  ready in 1234 ms

  ➜  Local:   http://localhost:5173/
  ➜  Network: use --host to expose
  ➜  press h + enter to show help
```

**此时**：
- ✅ 网站已启动
- ✅ 访问 `http://localhost:5173` 即可看到网站
- ✅ **不要关闭这个命令窗口**，关闭后网站就停止了

## ❌ **常见问题**

### **Q1: 提示"npm不是内部或外部命令"？**
**A**: 您还没有安装Node.js
1. 访问：https://nodejs.org/
2. 下载并安装LTS版本
3. 安装完成后**重启电脑**
4. 再次尝试

---

### **Q2: 提示"端口5173已被占用"？**
**A**: 可能已经启动过一次了
1. 检查是否有其他命令窗口在运行
2. 或者按 `Ctrl + C` 停止当前进程
3. 再次运行 `npm run dev`

---

### **Q3: npm install很慢或卡住？**
**A**: 切换到国内镜像源
```
npm config set registry https://registry.npmmirror.com
npm install
```

---

### **Q4: 启动后浏览器显示空白页？**
**A**: 
1. 按 `F12` 打开开发者工具
2. 查看Console标签是否有错误
3. 尝试刷新页面（`Ctrl + F5`）

---

### **Q5: 如何停止网站？**
**A**: 
1. 在命令窗口按 `Ctrl + C`
2. 或直接关闭命令窗口

## 🎬 **视频教程式说明**

### **第一幕：安装Node.js**
1. 打开浏览器
2. 访问 https://nodejs.org/
3. 点击绿色的"LTS"下载按钮
4. 下载完成后双击安装
5. 一路"下一步"，默认设置即可
6. 安装完成后**重启电脑**

### **第二幕：打开项目**
1. 按 `Win + R`
2. 输入 `cmd`，按回车
3. 输入 `cd ` （注意有个空格）
4. 把项目文件夹拖到命令窗口
5. 按回车

### **第三幕：安装依赖（仅第一次）**
1. 输入：`npm install`
2. 按回车
3. 等待几分钟（看到进度条）

### **第四幕：启动网站**
1. 输入：`npm run dev`
2. 按回车
3. 等待几秒
4. 看到 `Local: http://localhost:5173/`

### **第五幕：打开网站**
1. 打开浏览器（Chrome/Edge/Firefox）
2. 在地址栏输入：`http://localhost:5173`
3. 按回车
4. 🎉 看到网站首页！

## ⚡ **一键启动脚本（高级）**

如果您经常需要启动网站，可以创建一个快捷方式：

### **Windows批处理文件**

1. 在项目文件夹右键 → 新建 → 文本文档
2. 命名为 `启动网站.bat`
3. 右键编辑，输入：
```batch
@echo off
echo 正在启动网站...
npm run dev
pause
```
4. 保存
5. 以后双击 `启动网站.bat` 即可一键启动
